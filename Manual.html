Overview

The Course Registration System allows students to enroll in courses, view schedules, and manage registrations.

Web Pages Summary

  • Home Page: Login and introduction.
  • Course Listings: View available courses.
  • Registration Page: Register for courses.
  • Schedule Page: View registered courses.

Detailed Page Descriptions

Registration Page

The registration page contains the following fields:

  • Student ID: Enter numeric values (e.g., 123456).
  • Course Code: Enter course codes (e.g., CS101).
